## Project Overview

This project is a Machine Learning-based Credit Scoring System that predicts whether a customer is financially trustworthy or risky based on historical financial data.

The system uses classification algorithms such as:
- Logistic Regression
- Decision Tree
- Random Forest

The goal of this project is to help automate loan approval and credit risk assessment in the banking and finance sector.

---

##   Features

- Data preprocessing and cleaning
- Handling missing values
- Encoding categorical variables
- Exploratory Data Analysis (EDA)
- Data visualization using graphs and heatmaps
- Training multiple ML classification models
- Model evaluation using:
  - Accuracy
  - Precision
  - Recall
  - F1-Score
  - Confusion Matrix
- Comparison of model performance

##  Dataset

The project uses the German Credit Dataset containing customer financial information such as:
- Age
- Sex
- Job
- Housing
- Saving Accounts
- Checking Account
- Credit Amount
- Loan Duration
- Purpose
- Risk

Dataset Source:
https://www.kaggle.com/datasets/kabure/german-credit-data-with-risk

##  Machine Learning Models Used

### 1. Logistic Regression
A basic classification algorithm used as a baseline model.

### 2. Decision Tree
A tree-based algorithm that learns decision rules from data.

### 3. Random Forest
An ensemble learning algorithm that combines multiple decision trees for improved accuracy.
